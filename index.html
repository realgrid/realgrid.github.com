RealGrid를 경험해 보세요.

RealGrid는 높은 개발 생산성과 편리한 사용성을 제공하면서 탁월한 성능도 갖추고 있습니다.
데모사이트에서 RealGrid를 경험해 볼 수 있습니다.
좀더 자세한 테스트를 원하시면 평가판 요청 페이지에서 평가판을 신청하세요.

기초과정 튜토리얼

기초과정 튜토리얼에서는 RealGrid를 처음 접하는 사용자를 위한 손쉬운 연습을 제공합니다. 설치하는 방법 부터 데이터를 조작하는 방법까지 하나하나 따라하면서 배울수 있게 구성되어 있습니다.

API 레퍼런스

API 레퍼런스는 RealGrid가 가진 API에 대해 기본적인 설명을 제공합니다. 데모사이트나 튜토리얼에서 해당 기능으로 링크가 되어 있으며, 손쉽게 이름을 검색해 볼 수 있습니다.

도움 요청 메일

RealGrid로 프로젝트를 진행하는 중 도움이 필요하시면 support@realgrid.com 으로 기술지원 요청 메일을 보내세요. 뛰어난 실력을 갖춘 기술지원팀 요원이 답변을 보내드리고 문제가 해결 될때까지 도움을 드립니다.
