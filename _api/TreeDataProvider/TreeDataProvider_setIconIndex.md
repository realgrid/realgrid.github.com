#### Description

> 지정한 데이터행의 아이콘을 변경한다.   

#### Syntax

> function setIconIndex(rowId, iconIndex)   

#### Parameters

> **rowId**   
> Type: Number   
> 아이콘을 변경할 데이터행의 트리 아이디.   

> **iconIndex**   
> Type: Number   
> 새로 변경할 아이콘 인덱스.   

#### Return value

> None.   

#### Example

<pre class="prettyprint">
    tree.setIconIndex(1, 2);
</pre>
