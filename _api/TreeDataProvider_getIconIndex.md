#### Description

> 트리 행에 표시된 아이콘 인덱스를 가져온다.

#### Syntax

> function getIconIndex(rowId)

#### Parameters

> *rowId*
> Type: Number
> 데이터 행의 트리 아이디.

#### Return value

> Type: Number
> 아이콘 인덱스.

#### Example

<pre class="prettyprint">
    var icon = tree.getIconIndex(10);
</pre>
