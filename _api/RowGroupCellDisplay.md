> 그리드 행이 그룹 상태일 때 셀들의 표시 방법을 정의합니다.

#### Members

> *NORMAL*   
> Value: "normal"   
> 그룹핑되지 않은 컬럼처럼 모든 row의 셀들을 표시한다.

> *MERGE* 
> Value: "merge" 
> 해당 그룹의 모든 row의 셀들을 하나의 대표 셀로 표시한다.

> *HIDE* 
> Value: "hide" 
> 해당 컬럼 자체를 표시하지 않는다.
