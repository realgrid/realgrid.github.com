> 그리드 헤더 영역에 표시되는 컬럼 헤더에 대한 설정 정보를 관리한다.

#### Properties

> *text*
> Type: String 
> Default: null    
> 컬럼 헤더에 표시될 문자열을 지정한다. 이 값이 null 이면 컬럼에 연결된 데이터필드 이름을 표시한다. 데이터필드가 연결되지 않았다면 컬럼의 이름을 표시한다.
