#### Description

> Check된 데이터 행들의 목록을 반환한다.

#### Syntax

> function getCheckedRows()

#### Parameters

> None.

#### Return value

> Type: Array of Number
> Check된 데이터행들의 트리 아이디의 배열.

#### Examples 

<pre class="prettyprint">
    var rows = treeView.getCheckRows();
</pre>
