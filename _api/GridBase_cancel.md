#### Description

> 현재 편집 중인 행의 편집을 취소한다. 편집 중이 아니라면 아무 일도 하지 않는다. 편집을 완료할 때는 [commit|commit()](/api/GridBase/)을 호출한다.

#### Syntax

> function cancel()

#### Parameters

#### Return value

None

#### Example

<pre class="prettyprint">
    grid.cancel();
</pre>
