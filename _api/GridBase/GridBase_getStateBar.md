#### Description

> 그리드 StateBar 설정 정보를 가져온다. [StatusBar](/api/types/StatusBar/) 가 설정 모델이다.

#### Syntax

> function getStateBar()

#### Parameters

> None

#### Return value

> Type: object  
> [StatusBar](/api/types/StatusBar/) 설정 모델과 동일한 내용의 객체가 반환된다.

#### Example

<pre class="prettyprint">
var bar = grid.getStateBar();
</pre>
