#### Description

> 지정된 SourceId에 해당하는 lookup source를 clear한다.

#### Parameters

> **sourceId**  
> Type: String  
> LookupSource의 Id  

#### Return value

> None.  

#### Example

<pre class="prettyprint">
    gridView.clearLookupData(sourceId)
</pre>
