#### Description

> checkBar.head 영역에 v 표시 활성화 여부를 지정한다.  
> checkbar에 체크 표시되는 것은 아니다.  

#### Syntax

> function setAllCheck(checked)  

#### Parameters

> **checked**  
> Type: Boolean  
> 체크 표시 활성화 여부  

#### Return value

> None.

#### Examples 

<pre class="prettyprint">
	gridView.setAllCheck(true);
</pre>
