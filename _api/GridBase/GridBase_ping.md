#### Description

> RealGrid+ Web의 경우 입력된 메세지를 그리드 대화상자에 표시한다.  
> RealGridJS의 경우 alert창으로 입력된 메세지를 표시한다.  

#### Syntax

> function ping(message)

#### Parameters

> **message**  
> Type: String  
> 입력된 메세지를 그리드 대화상자에 표시한다.  

#### Return value

> none

#### Examples 

<pre class="prettyprint">
    gridView.ping("테스트");
</pre>
