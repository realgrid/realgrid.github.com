#### Description

그리드 맨 위에 지정한 행이 위치한다.         

#### Syntax

> function setTopItem(value)

#### Parameters

> **value**  
> Type: Number  
> 그리드 상단에 위치하고자하는 행을 지정한다.    

#### Return value

> None

#### Examples 

<pre class="prettyprint">
    gridView.setTopItem(20)
</pre>
