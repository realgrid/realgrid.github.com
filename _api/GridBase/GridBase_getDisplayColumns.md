#### Description

 현재 보여지고 있는 컬럼들의 컬럼명을 가져온다.  

#### Syntax

> function getDisplayColumns()

#### Parameters

> None.    

#### Return value

> Type: Array of String  
> 현재 보여지고 있는 컬럼명의 배열    

#### Examples 

<pre class="prettyprint">
gridView.getDisplayColumns();
</pre>
