#### Description

> 그리드 Footer와 관련된 설정 정보들을 가져온다. [Footer](/api/GridBase/)가 설정 모델이다.

#### Syntax

> function getFooter()

#### Parameters

> None

#### Return value

> Type: object
> [Footer](/api/GridBase/) 설정 모델과 동일한 내용의 객체가 반환된다.

#### Example

<pre class="prettyprint">
var footer = grid.getFooter();
</pre>
