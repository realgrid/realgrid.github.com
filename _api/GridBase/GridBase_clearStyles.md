#### Description

> 지정한 영역의 스타일 값들을 모두 제거해서 기본값을 따르게 한다.  

#### Syntax

> function clearStyles(region)  

#### Parameters

> **region**  
> Type: string  
> 영역 이름  


#### Return value

> None.

#### Example

<pre class="prettyprint">
    gridView.clearStyles("header");
</pre>
