#### Description

> 현재 위치한 셀에서 그리드의 editor 를 보여준다.
> 그리드가 편집 불가 상태(editable: false)라면 editor는 표시되지 않는다.

#### Syntax

> function showEditor()

#### Parameters

> None

#### Return value

> None

#### Example

<pre class="prettyprint">
    grdMain.showEditor();
</pre>
