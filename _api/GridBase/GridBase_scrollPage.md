#### Description

 아이템 모델의 부모 아이템 모델을 리턴합니다.    

#### Syntax

> function scrollPage(delta)    

#### Parameters

> **delta**    
> Type: Number    
> 현재를 기준으로 이동하려는 페이지 수를 지정한다. 이전 페이지로 가려면 음수로 입력한다.       

#### Return value

> None.  

#### Examples 

<pre class="prettyprint">
gridView.scrollPage(2);
</pre>
