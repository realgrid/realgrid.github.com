#### Description

> RealGridJS가 사용하는 Context들의 Root 경로를 지정한다.

#### Syntax

> function setRootContext(path)  

#### Parameters

> **path**    
> Type: String    
> 경로를 지정한다.    

#### Return value

> None.  

#### Example

<pre class="prettyprint">
    RealGridJS.setRootContext("/img/realgridjs");
</pre>
