#### Description

> 그리드 내부에서 발생하는 각종 이벤트들의 log정보를 console창에 표시한다.

#### Syntax

> function setTrace(enable)  

#### Parameters

> **enable**    
> Type: Boolean    
> 표시 여부를 지정한다.    

#### Return value

> None.  

#### Example

<pre class="prettyprint">
    RealGridJS.setTrace(true);
</pre>
