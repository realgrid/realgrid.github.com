#### Description

> 지정한 itemIndex의 CheckBar에 checkable 상태를 지정한다.

#### Syntax

> function setCheckBar(options)

#### Parameters

> **itemIndex**
> Type: Number
> Grid Item의 index이다.

> **value**
> Type: Boolean
> Checkable 상태

#### Return value

> None.

#### Example

<pre class="prettyprint">
    grid.setCheckable(1, true);
</pre>
