#### Description

> 부모행의 itemIndex를 가져온다.

#### Syntax

> function getParent(itemIndex)

#### Parameters

> *itemIndex*
> Type: Number
> 자식행의 itemIndex를 입력한다.

#### Return value

> Type: Number
> 부모행의 itemIndex이다.

#### Example

<pre class="prettyprint">
    treeMain.getParent(2);
</pre>
