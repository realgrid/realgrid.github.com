#### Description

> 입력된 메세지를 그리드 대화상자에 표시한다.

#### Syntax

> function ping(message)

#### Parameters

> *message*
> Type: String
> 입력된 메세지를 그리드 대화상자에 표시한다.

#### Return value

> none

#### Example

<pre class="prettyprint">
    grdMain.ping("테스트");
</pre>
