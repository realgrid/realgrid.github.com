#### Description

 그리드의 Indicator영역에 표시하는 방법을 정의합니다.

#### Members

> **NONE**    
> Value: "none"    
> 표시하지 않습니다.                                          

> **INDEX**  
> Value: "index"   
> 1부터 시작하여 표시합니다. (정렬에 무관하게 1번부터 시작) 

> **ROW**     
> Value: "row"    
> Row의 고유번호를 표시합니다. (정렬에 따라서 섞여서 표시)
