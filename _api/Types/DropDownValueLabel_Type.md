#### Description

 DropDown Editor의 목록 표시 형태를 지정한다.     

#### Members

> **VALUE**    
> Value: "value"     
> value만 표시된다.                                                   

> **LABEL**  
> Value: "label"    
> label만 표시된다.     

> **VALUE_LABEL**     
> Value: "valueLabel"    
> value와 label의 순서로 둘 다 표시된다.        

> **LABEL_VALUE**     
> Value: "labelValue"    
> label과 value의 순서로 둘 다 표시된다.
