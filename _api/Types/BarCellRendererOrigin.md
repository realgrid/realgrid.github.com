#### Description

> Bar Cell Renderer에서 표시되는 막대상자의 시작 위치를 지정합니다.

#### Members

> **DEFAULT**       
> Type: "default"       
> 기본값, 미지정시 BarCellRendererOrigin.LEFT 와 같습니다.      

> **LEFT**      
> Type: "left"       
> 막대 상자가 셀의 왼쪽부터 사작한다. 

> **RIGHT**      
> Type: "right"       
> 막대 상자가 셀의 오른쪽부터 사작한다. 

> **TOP**  
> Type: "top"   
> 막대 상자가 셀의 위쪽부터 사작한다.

> **BOTTOM**  
> Type: "bottom"  
> 막대 상자가 셀의 아래쪽부터 사작한다.
