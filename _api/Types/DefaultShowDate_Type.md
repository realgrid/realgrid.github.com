#### Description

 Date Editor 팝업시 표시되는 날짜를 정의합니다.

#### Members

> **NORMAL**    
> Value: "normal"    
> 셀에 data가 있는 경우 해당일자, 없는 경우 이전에 popup된 날짜 또는 현재일자가 선택되어 표시됩니다.                                                

> **TODAY**  
> Value: "today"   
> 현재 일자가 선택되어 표시됩니다.  

> **TODAY_WHEN_NULL**     
> Value: "todayWhenNull"    
> "normal"의 동작과 같고 날짜가 없는 경우에만 현재 일자가 선택되어 표시됩니다.
