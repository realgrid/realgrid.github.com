#### Description

 마우스 호버링시 그리드에 표시되는 행 선택 영역 방식을 정의한다.

#### Members

> **ROW**  
> Type: "row"    
> indicator, stateBar, checkBar, column을 포함한 영역            

> **DATA**    
> Type: "data"   
> column 영역만              

> **FILL** 
> Type: "fill"    
> indicator, stateBar, checkBar, column을 포함한 영역 + empty 영역               

> **CELL**    
> Type: "cell"   
> 하나의 셀 영역
