#### Description

정렬시 대소문자 구분 여부.  

#### Members

> **SENSITIVE**  
> Value: "sensitive"  
> 정렬시 대소문자를 다른문자로 취급한다.  

> **INSENSITIVE**   
> Value: "insensitive"   
> 정렬시 대소문자를 같은문자로 취급한다.
