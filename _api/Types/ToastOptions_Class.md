#### Description

대량 데이터셋의 정렬/필터링/그룹핑시 화면이 멈춰있는 상태일때 진행중임을 알려주는 Toast View 표시여부와 메세지를 지정한다.


#### Properties

> **visible**  
> Type: boolean    
> Default: false  
>

> **message**  
> Type: string  
> Default: 지정되지 않으면 sorting/filtering/grouping 별로 기본값이 표시
