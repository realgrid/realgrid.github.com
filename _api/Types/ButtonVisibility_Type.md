#### Description

> 데이터 셀에 버튼을 표시하는 방법을 지정한다.

#### Members

> **ALWAYS**   
> Value: "always"  
> 항상 버튼을 표시한다.   

> **DEFAULT**  
> Value: "default"   
> hovering, focused상태에서만 버튼을 표시한다.    

> **VISIBLE**  
> Value: "visible"    
> focused된 상태에서만 버튼을 표시한다.       

> **HIDDEN**   
> Value: "hidden"  
> 버튼을 표시하지 않는다.
