#### Description

> 컬럼 필터링에 관련된 설정 정보를 관리한다.

#### Properties

> **name**  
> Type: String   
> Default:  null     
> 필터 이름.  

> **criteria**  
> Type: String  
> Default: null    
> 필터에 표시될 [Expression](/api/features/Expression)을 지정한다.  

> **text**  
> Type: String  
> Default: null    
> selector에 표시되는 문자열이다. null이면 name 이 표시된다.  

> **description**  
> Type: String   
> Default:  null     
> 필터의 설명을 기재한다.  

> **active**  
> Type: Boolean   
> Default: true     
> 필터를 수행할 것인 지를 지정한다.
