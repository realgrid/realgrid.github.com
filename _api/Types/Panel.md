#### Description

> 그리드 Panel 표시 정보들에 대한 설정 모델이다.

#### Properties

> **minHeight**  
> Type: number  
> Default: 20   
> 그리드 Panel의 최소 높이를 픽셀 단위로 지정한다. height가 0이면 이 높이가 Panel의 높이를 결정한다. 

> **visible**    
> Type: boolean   
> Default: true   
> true면 그리드 Panel을 표시한다.

> **height**  
> Type: number  
> Default: 0  
> 그리드 Panel의 높이를 픽셀 단위로 지정한다.
