#### Description

> 아이콘을 표시할 위치. "left", "right", "top", "bottom", "center" 중 하나. 기본값은 "left" 이며. 아이콘 위치에 따라 텍스트의 위치도 변경된다.  
iconAlignment, iconPadding, iconOffset 등과 함께 사용한다.
  
#### Members

> **LEFT**  
> Value: "left"  
> Icon을 Text 왼쪽에 표시한다.  

> **RIGHT**  
> Value: "right"  
> Icon을 Text 오른쪽에 표시한다.  

> **TOP**  
> Value: "top"  
> Icon을 Text 위에 표시한다.  

> **BOTTOM**   
> Value: "bottom"  
> Icon을 Text 아래 표시한다.  

> **CENTER**   
> Value: "center"  
> Icon을 Cell의 중앙에 표시한다. 이때는 Text가 표시되지 않는다.
