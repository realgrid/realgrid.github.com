> Group컬럼에서 하위 컬럼을 배치하는 방향을 정의합니다.

#### Members

> **HORIZONTAL** 
> Value: "horizontal" 
> 하위 컬럼들을 가로 방향으로 배치합니다. 

> **VERTICAL**   
> Value: "vertical"   
> 하위 컬럼들을 세로 방향으로 배치합니다.
