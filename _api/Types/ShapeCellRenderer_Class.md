#### Description

> 그리드에 포함되어 있는 몇 개의 shape 아이콘 중 하나를 텍스트와 같이 표시한다.   
Shape는 컬럼 스타일의 figureSize 설정으로 크기를 자유롭게 지정할 수 있고, figureBackground로 색상 또한 다양하게 설정할 수 있다. Shape 타입은 figureName 속성으로 지정한다.

#### Properties

> None.
