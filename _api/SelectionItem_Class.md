> 그리드 선택 영역 정보 모델.

#### Properties

> *style*
> Type: string 
> 선택 형태. "block", "rows", "columns" 중 하나다.

> *startItem*
> Type: number 
> 시작 행의 아이템 인덱스.

> *startRow* 
> Type: number       
> 시작 행의 데이터 행 번호. 

> *startColumn* 
> Type: string 
> 시작 컬럼의 이름.   

> *endItem*
> Type: number 
> 끝 행의 아이템 인덱스.

> *endRow* 
> Type: number       
> 끝 행의 데이터 행 번호. 

> *endColumn* 
> Type: string 
> 끝 컬럼의 이름.
