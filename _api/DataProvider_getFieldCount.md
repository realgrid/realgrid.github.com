#### Description

> Data provider에 설정된 데이터필드의 개수를 가져온다.

#### Syntax

> function getFieldCount()

#### Parameters

> None.

#### Return value

> Type: number
> 필드 개수.

#### Example

<pre class="prettyprint">
    var count = provider.getFieldCount();
</pre>
