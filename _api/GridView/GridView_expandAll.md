#### Description

 모든 그룹을 펼친다. 레벨을 지정하면 그 레별까지 펼쳐진다.

#### Syntax

> function expandAll(level)

#### Parameters

> **level**  
> Type: number  
> Default: 0  
> 펼치고 싶은 레벨을 지정한다. 

#### Return value

> None.

#### Examples 

<pre class="prettyprint">
	gridView.expandAll();
</pre>
