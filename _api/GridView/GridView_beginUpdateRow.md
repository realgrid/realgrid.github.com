#### Description

 입력된 itemIndex에 해당하는 데이터 행을 편집상태로 변경한다.

#### Syntax

> function beginUpdateRow(itemIndex)

#### Parameters

> **itemIndex**  
> Type: Number  
> Default: -1  

#### Return value

> None.

#### Examples 

<pre class="prettyprint">
    gridView.beginUpdateRow(0);
</pre>
