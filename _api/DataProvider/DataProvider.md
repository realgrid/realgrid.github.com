#### Description

> 그리드나 트리에 표시되는 원본 데이터를 보관하고 있는 Data source 객체를 위한 기반 클래스다. [LocalDataProvider](/api/LocalDataProvider/)와 [TreeDataProvider](/api/TreeDataProvider/)가 이 클래스를 계승한다.
