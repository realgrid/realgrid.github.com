#### Description

> 트리뷰의 모든 아이템에 체크 또는 해제한다.

#### Syntax

> function checkAll(checked)

#### Parameters

> **checked**
> Type: boolean
> Default: true
> 입력하지 않으면 true이며 모든 아이템을 체크한다. false로 입력하면 모든 아이템의 체크 박스를 해제한다.

#### Return value

> None.

#### Example

<pre class="prettyprint">
    treeMain.checkAll(true);
</pre>
