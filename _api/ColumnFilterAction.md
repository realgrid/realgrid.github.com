> 컬럼 필터 액션에 관련된 설정 정보를 관리한다.

#### Properties

> **name**
> Type: String 
> Default:  null    
> 필터 이름. 

> **text**
> Type: String 
> Default: null   
> selector에 표시되는 문자열이다. null이면 name 이 표시된다.

> **description**
> Type: String 
> Default:  null    
> 필터의 설명을 기재한다.
