#### Description

 피벗 그리드에 설정된 컨택스트 메뉴 정보를 가져온다.      

#### Syntax

> function getGlobalContextMenu()  

#### Parameters

> None.

#### Return value

> Type: Object  
> text와 callback으로 구성된 메뉴 정보 객체 배열.  

#### Examples 

<pre class="prettyprint">
var contextMenu = pivot.getGlobalContextMenu();
</pre>
