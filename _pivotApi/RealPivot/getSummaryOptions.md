#### Description

 피벗 그리드의 Summary옵션정보들을 가져온다.    

#### Syntax

> function getSummaryOptions()  

#### Parameters

> None.

#### Return value

> Type: Object   
> [SummaryOptions](/pivotApi/types/SummaryOptions/)들로 구성된 객체를 반환한다.  

#### Examples 

<pre class="prettyprint">
    var options = pivot.getSummaryOptions()
</pre>
