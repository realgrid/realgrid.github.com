#### Description

 피벗 그리드에 설정된 Display옵션정보들을 가져온다.    

#### Syntax

> function getDisplayOptions()  

#### Parameters

> None.

#### Return value

> Type: Object   
> [DisplayOptions](/pivotApi/types/DisplayOptions/)들로 구성된 객체를 반환한다.     

#### Examples 

<pre class="prettyprint">
var options = pivot.getDisplayOptions();
</pre>
