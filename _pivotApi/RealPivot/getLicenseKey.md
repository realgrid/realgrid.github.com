#### Description

RealPivot객체에 적용되어있는 licenseKey를 출력한다.    

#### Syntax

> function getLicenseKey()  

#### Parameters

> None.    

#### Return value

> Type: string    
> [setLicenseKey](/pivotApi/RealPivot/setLicenseKey/)를 이용해서 등록한 licensekey 또는 window에 설정된 licensekey를 출력한다.

#### Examples 

<pre class="prettyprint">
console.log(RealPivot.getLicenseKey());
</pre>
