#### Description

RealPivot객체에 licensekey를 입력한다.    
환경변수에 저장된 licensekey를 RealPivot객체에 적용할때 사용한다.    

#### Syntax

> function setLicenseKey(licensekey)  

#### Parameters

> **licensekey**   
> Type: string   
> licensekey를 지정한다. 


#### Return value

> None
