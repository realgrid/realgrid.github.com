#### Syntax

> function getSortOptions()  

#### Parameters

> None.

#### Return value

> Type: Object   
> [SortOptions](/pivotApi/types/SortOptions/)들로 구성된 객체를 반환한다.     

#### Examples 

<pre class="prettyprint">
var options = pivot.getSortOptions();
