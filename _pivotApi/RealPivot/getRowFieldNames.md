#### Description

 피벗 그리드에 표시되고 있는 행 필드의 이름들을 가져온다.      

#### Syntax

> function getRowFieldNames()  

#### Parameters

> None.

#### Return value

> Type: Array of String   
> 행 필드의 이름들을 가져온다.     

#### Examples 

<pre class="prettyprint">
var rNames = pivot.getRowFieldNames();
</pre>
