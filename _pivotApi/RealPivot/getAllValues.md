#### Description

 피벗 그리드에 표시되고 있는 값들을 계층 구조로 가져온다.        

#### Syntax

> function getAllValues()  

#### Parameters

> None.

#### Return value

> Type: Object
> 피벗그리드의 값들을 계층 구조로 가져온다.       

#### Examples 

<pre class="prettyprint">
var values = pivot.getAllValues();
</pre>
